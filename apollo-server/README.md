# Tiny LionGarage middleware

Tiny node server using apollo-server to fetch static data from a json 

The used Nodejs version is `14.16.1`.

Frameworks used:
* apollo-server;
* express;  
* graphql;
* Jest;

## Install

Run `npm i`.

The used npm version is `6.14.12`.

## Run server

Run `npm start` to start up the server.

## Run test

Run `npm test`.
